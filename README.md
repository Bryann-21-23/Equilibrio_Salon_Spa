# 🌿 Equilibrio - Salón & Spa
### Sistema de Gestión Inteligente y Control Operativo

Bienvenido a **Equilibrio**, una plataforma diseñada a medida para la gestión administrativa y financiera de un salón de belleza y spa moderno. Este proyecto no es solo un software; es el resultado de un flujo de trabajo de **Desarrollo asistido por IA**, donde la visión estratégica humana y la precisión técnica artificial se unieron para crear una herramienta robusta en tiempo récord.

---

## 🚀 Filosofía de Construcción: Humano + IA
Este sistema demuestra un nuevo estándar de ingeniería:
- **Supervisión Humana:** Cada funcionalidad, desde el sistema de IDs de clientes hasta la paleta de colores "Verde Equilibrio", fue definida y supervisada paso a paso.
- **Agentes de IA:** Se utilizaron agentes avanzados para realizar cambios quirúrgicos en el código, conectar bases de datos en la nube y optimizar el rendimiento.
- **Iteración Ágil:** El proyecto evolucionó desde un almacenamiento local simple hasta una infraestructura profesional en la nube (Supabase) y despliegue continuo (Vercel).

---

## 💎 Funcionalidades Clave

### 👤 Gestión de Clientes (CRM)
- **Registro Inteligente:** Captura de datos esenciales (Nombre, Celular, Correo, Cumpleaños).
- **ID Único Automático:** Sistema generador de códigos de 5 caracteres (ej. `4821K`) para una identificación rápida y comercial.
- **Ficha Interactiva:** Lista de usuarios expandible mediante un diseño de acordeón elegante que muestra estadísticas de consumo y un historial completo de visitas.

### ✂️ Control de Servicios
- **Vinculación Inteligente:** Buscador en tiempo real que conecta servicios con clientes registrados automáticamente.
- **Registro Simplificado:** Interfaz optimizada para registrar ventas de forma rápida, incluso en dispositivos móviles.

### 📊 Análisis Financiero
- **Tablero Económico:** Visualización de ingresos totales, promedio por servicio y mejores días de venta.
- **Gráficas Dinámicas:** Seguimiento de ingresos en el tiempo con filtros avanzados por periodo (Día, Semana, Mes, Año) y por tipo de servicio específico.
- **Rankings:** Identificación instantánea de los servicios más pedidos y los clientes más leales.

### 🛡️ Centro de Control de Datos (Data-Ready para IA)
El sistema está preparado para el futuro de la inteligencia de negocios:
- **Exportación a Excel:** Reportes completos con pestañas separadas para Clientes, Servicios y Personal.
- **Backup para IA:** Generación de archivos JSON "limpios", diseñados específicamente para ser subidos a modelos de lenguaje (ChatGPT, Claude, etc.) y obtener análisis predictivos del negocio.

---

## 🛠️ Tecnología en la Nube
- **Frontend:** Angular 18 con una estética minimalista y profesional.
- **Base de Datos:** Supabase (PostgreSQL) para una sincronización en la nube 24/7.
- **Reportes:** Librería XLSX para procesamiento de datos de alta velocidad.

---

## 👨‍💼 Gestión de Personal
El sistema incluye un módulo de administración restringido donde se pueden dar de alta a nuevas usuarios, asignar roles y asegurar que el acceso a los datos económicos sea solo para el personal autorizado.
